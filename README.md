# Sistema de Calificaciones Academicas

Este proyecto es una calculadora de notas que permite a los estudiantes ingresar sus calificaciones y pesos correspondientes para calcular su promedio. Y determina si el estudiante aprueba o reprueba y asigna una calificacion

## Caracteristicas
- Ingreso de 5 notas y sus pesos
- Calculo del promedio
- (Aprobado/Reprobado)

## Uso
1. Ingresa el nombre del estudiante
2. Completa las 5 notas y sus pesos
3. Haz clic en el botón "Calcular Promedio" para ver los resultados
